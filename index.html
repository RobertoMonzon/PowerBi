Power BI Dashboards Portfolio (2025–2023)

Shape Map Dashboard (2025)

This interactive dashboard uses the Shape Map visual, a feature that needs to be enabled in Mexico, to identify regions with the highest customer concentration. It provides a dynamic geographical representation that allows users to visualize hotspots of client activity across the country. The use of custom shapes and layers makes the navigation and analysis experience highly engaging and intuitive.

Sales Routes with Azure Maps (2025)

This dashboard integrates Azure Maps to visualize and trace the real travel routes of sales representatives across Mexico. It combines geographical precision with interactive filters, allowing users to analyze visit coverage, territory efficiency, and performance trends. The map dynamically adapts as users select representatives or regions, offering a smooth, data-driven exploration.

Market Dashboard – CEESA Americas (2024)

My masterpiece dashboard for CEESA showcases advanced Power BI capabilities, including dynamic navigation menus, top 5 product visualizations, and performance toggles between YTD (Year-to-Date) and MAT (Moving Annual Total). It delivers a modern, interactive experience that helps users track growth, market performance, and product trends seamlessly. This project demonstrates my ability to combine data storytelling with design and technical optimization.

Interactive Survey Dashboard (2024)

This project showcases the interactive power of Power BI, where images change dynamically based on user menu selections. The dashboard also includes a histogram to visualize response distribution and a shape map to locate responses geographically. This design highlights how Power BI can be transformed into an interactive app-like experience, improving user engagement and data exploration.

Interactive Dashboard on Gastrointestinal Issues (2023)

An interactive dashboard simulating the gastrointestinal system, visualizing patient distributions, diagnoses, and case trends by month and gender. Built entirely in Power BI, it combines storytelling visuals with healthcare insights for a fully dynamic experience.

Interactive 3D Dental Dashboard (2023)

A 3D dental dashboard that allows users to explore teeth individually, view patient treatment data, and analyze performance by dentist. Includes interactive line charts, gender segmentation, and revenue tracking tools — all designed with clarity and engagement in mind.

Spinal Cord Injury Dashboard (2023)

A fully immersive dashboard that allows users to toggle between 2D and 3D modes, exploring different regions of the spine. It brings anatomy and analytics together, transforming medical data into an interactive educational experience.

Comprehensive Accident Report (2023)

A detailed Power BI report covering accident data throughout 2023. It includes gender and severity breakdowns, monthly and weekly trends, and interactive filters to analyze incidents by shift and area — offering a complete view of workplace safety performance.

Butcher Shop Performance Dashboard (2023)

A creative Power BI dashboard designed like a chalkboard in a butcher shop. It includes a cow-shaped interactive visual, financial KPIs, customer map of Mexico, branch comparisons, and dynamic performance messages — transforming data into a visually memorable experience.

Health & Safety Dashboards (2023–2024)

A pair of dashboards designed for corporate health and safety monitoring. The first includes navigation icons representing body parts to explore exam results interactively, while the second focuses on year-over-year accident comparisons. Both combine analytics, design, and functionality for HR and HSE teams.
